# Yapay Zeka Model Katmanı
Bu dizin, bitki hastalıklarını Edge AI tabanlı analiz edecek modelin (MobileNet V3) eğitim süreçlerinin yürütülmesi için ayrılmıştır.

- `datasets`: Çiğ ve etiketlenmiş yaprak lezyon verileri (Train / Validation setleri).
- `notebooks`: Model eğitim ve test raporlarının bulunduğu Jupyter (.ipynb) dosyaları.
- `tflite_export`: Mobile (.tflite vb.) aktarılacak hazır / nicemlenmiş (quantized) modeller.
